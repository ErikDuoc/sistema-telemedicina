# Payment Service

Microservicio Spring Boot para procesamiento de pagos médicos.

## Tecnologías

- Java 21
- Spring Boot 3.5
- Spring Data JPA
- H2 Database
- Maven

## Endpoints

### Procesar pago
POST /api/payments/process

### Listar seguros
GET /api/payments/insurances
