# Telemedicina System

Proyecto Maven Multi-Module para plataforma de telemedicina y gestión hospitalaria.

## Cómo abrir en IntelliJ

1. Descomprimir el ZIP
2. Abrir SOLO la carpeta:
   telemedicina-system

3. IntelliJ detectará automáticamente el proyecto Maven Multi-Module
4. Seleccionar:
   Load Maven Project

## Microservicios incluidos

- patient-service
- doctor-service
- notification-service
- payment-service
- agenda-service
- laboratory-service
- appointment-service
- clinical-record-service
- prescription-service
- video-consultation-service
